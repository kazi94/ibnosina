    Gestion de l'éducation thérapeutique

    La prescription des médicaments est un point essentiel pour la prise en charge de soin du patient. Hormis si le patient ne respecte pas les recommandations du médecin, le processus de rétablissement aboutira vers l’échec. Afin d’éviter tout désagrément, le pharmacien pratique l’éducation thérapeutique envers le patient et le forme à respecter la prise de médicament et à améliorer l’hygiène de vie quotidienne.
    La gestion de l'éducation thérapeutique est soumis à des droit d'accés pour chaque utilisateur. Voir Profils.

    Liste des éducations thérapeutiques faites

    • Cliquer sur le bouton pour supprimer l'éducation thérapeutique.
    • Cliquer sur le bouton pour exporter l'historique des éducations thérapeutiques faites.
    • Cliquer sur le bouton pour commenter (annotation) l'éducation thérapeutique.
    • Pour imprimer le rapport de l'éducation thérapeutique cliquer sur .
    • Pour visualiser la brochure cliquez sur

    Ajouter une éducation thérapeutique

    Cliquez sur le bouton Education therapeutique. Un popup va apparaitre pour ajouter une éducation thérapeutique.

    • Date Date de l'éducation thérapeutique.
    • Type type d'éducation thérapeutique faite par le pharmacien (formation/brochure).
    • Fichier Fichier à chargé tel qu'une brochure ou autre image.
    • Notes Commentaire sur l'éducation thérapeutique faite par le pharmacien.

    Liste des éducations thérapeutiques à faire

    Si la prescription envoyé par le médecin au pharamcien est à risque selon les règle de l'éducations thérapeutiques. Le pharmacien doit pratiquer l'éducation thérapeutique envers le patient. Voir Règles éducation thérapeutique.

    • Le bouton Détails affiche une fiche complète pour procéder à l'éducation thérapeutique. Voir Fiche éducation thérapeutique.
