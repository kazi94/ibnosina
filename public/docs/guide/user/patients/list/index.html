    Liste des patients

    Cette interface affiche la liste des patients hospitalisés dans le service

    • Cliquer sur l'icone pour accéder au dossier médicale du patient.
    • le bouton ajouter patient permet de renseigner un nouveau patient. Voir Ajouter patient.
    • Pour renseigner une hospitalisation ou une consultation cliquer sur le bouton Hospitalisation ou Consultation. Plus de détails Ajouter patient.

    Trier par colonnes

    Cliquez sur l'en-tête de n'importe quelle colonne pour trier les patients selon cette colonne.

    Filtrer les patients

    Tapez dans la barre de recherche search pour rechercher un élément dans le tableau.
    La recherche s'effectue sur toute les colonnes d'ou leurs contenus corresponds au terme recherché.

    Archives

    Pour afficher la liste des patients décédés ou sortie de l'hopital ou du service cliquer sur le bouton Archives.

    Conseil

    Si la barre de recherche ne s'affiche pas correctement veuillez actualiser la page.

    Vue grille

    Pour accéder à la vue grille cliquer sur le bouton Vue grille

    La vue grille affiche les photos des patients. il est possible d'accéder au dossier médicale du patient en cliquant sur la photo.

    Par défault la liste affiche les patients hospitalisés dans le service.

    • Tous permet d'afficher tout les patients.
    • Consultation permet d'afficher la liste des patients qui ont passé une consultation sans hospitalisation.
    • En Service permet d'afficher les patients du service.
    • Décédés permet d'afficher les patients décédés.
    • Hors chu permet d'afficher les patients sortie de l'hopital ou du service en cours.

    Astuce

    Sélectionner nom ou age pour ordonner la liste des patients selon cette option.
